### Kết quả đạt được tuần 12:

* Hoàn thành toàn bộ hệ thống Website Đặt Vé Sự Kiện.

* Kiểm thử thành công quy trình đặt vé:
  * Frontend gửi yêu cầu đến API Gateway.
  * API Gateway chuyển dữ liệu vào Amazon SQS.
  * AWS Lambda xử lý yêu cầu đặt vé.
  * Amazon RDS cập nhật dữ liệu đặt vé.

* Thực hiện kiểm thử tải (Stress Test) và theo dõi kết quả bằng Amazon CloudWatch.

* Hoàn thiện tài liệu hướng dẫn triển khai và báo cáo đồ án.

* Tổng kết kết quả thực hiện, đánh giá hệ thống và chuẩn bị cho buổi báo cáo bảo vệ đồ án.
