### Week 3 Objectives

* Learn AWS networking architecture using Amazon VPC.
* Deploy Amazon RDS in a private subnet.
* Connect Amazon EC2 to Amazon RDS, perform backup and restore operations, and clean up AWS resources.

* Gained a solid understanding of AWS networking architecture using Amazon VPC.

* Successfully created an Amazon VPC with Public and Private Subnets, and configured the Internet Gateway and Route Tables.

* Configured Security Groups for Amazon EC2 and Amazon RDS by following the Principle of Least Privilege.

* Successfully created a DB Subnet Group and deployed an Amazon RDS (MySQL) instance in a Private Subnet.

* Successfully connected an Amazon EC2 instance to Amazon RDS using the MySQL Client.
