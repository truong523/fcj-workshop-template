### Week 1 Achievements

* Gained a fundamental understanding of AWS and its core service categories:
  * Compute
  * Storage
  * Networking
  * Database

* Successfully created an AWS Free Tier account and accessed the AWS Management Console.

* Configured AWS Budgets to monitor and receive alerts for cloud spending.

* Became familiar with AWS Management Console and AWS CLI:
  * Installed AWS CLI.
  * Configured Access Key, Secret Access Key, and the default AWS Region.
  * Executed basic AWS CLI commands to verify the configuration.

* Learned the fundamentals of Amazon EC2:
  * Instance Types
  * Amazon Machine Images (AMI)
  * Elastic Block Store (EBS)
  * Security Groups
  * Elastic IP
  * SSH Connection Methods

* Studied Amazon RDS, including:
  * Supported Database Engines
  * Storage Options
  * Multi-AZ Deployment
  * Read Replicas
  * Backup and Restore
  * Security Features

* Acquired the ability to deploy a basic AWS environment and understand the workflow of provisioning AWS resources.

* Successfully completed all planned objectives for the first week and prepared for more advanced AWS hands-on activities in the following weeks.
