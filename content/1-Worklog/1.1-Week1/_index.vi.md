### Kết quả đạt được tuần 1:

* Hiểu được kiến thức nền tảng về AWS và các nhóm dịch vụ chính:
  * Compute
  * Storage
  * Networking
  * Database

* Tạo thành công tài khoản AWS Free Tier và đăng nhập được vào AWS Management Console.

* Thiết lập AWS Budgets để theo dõi và cảnh báo chi phí sử dụng.

* Làm quen với AWS Management Console và AWS CLI:
  * Cài đặt AWS CLI.
  * Cấu hình Access Key, Secret Access Key và Region mặc định.
  * Kiểm tra cấu hình và thực hiện các lệnh CLI cơ bản.

* Nắm được kiến thức cơ bản về Amazon EC2:
  * Instance Types.
  * Amazon Machine Images (AMI).
  * Elastic Block Store (EBS).
  * Security Groups.
  * Elastic IP.
  * Các phương thức kết nối SSH.

* Tìm hiểu Amazon RDS:
  * Các Database Engine được hỗ trợ.
  * Các loại Storage.
  * Multi-AZ.
  * Read Replica.
  * Backup và Restore.
  * Các cơ chế bảo mật.

* Có khả năng triển khai môi trường AWS cơ bản và hiểu quy trình tạo tài nguyên trên AWS.

* Hoàn thành toàn bộ mục tiêu đề ra trong tuần đầu tiên và sẵn sàng thực hành triển khai các dịch vụ AWS trong các tuần tiếp theo.
