#### Overview

In this section, you will build the **Frontend Website** for the Event Ticket Booking System. The frontend provides an intuitive user interface that allows users to browse available events, view ticket information, and submit ticket booking requests through a responsive web application.

The frontend is developed using the following technologies:

- **HTML5:** Defines the structure and content of the web pages.
- **CSS3:** Creates a modern, responsive, and visually appealing user interface.
- **JavaScript:** Handles client-side interactions, form validation, countdown timers, and communication with backend APIs.
- **Amazon API Gateway:** Receives booking requests from the frontend and forwards them to the backend Serverless architecture.
- **Amazon SES:** Sends confirmation emails to users after their booking has been successfully processed by the backend.

In this chapter, you will learn how to build the homepage, create the ticket booking interface, connect the frontend to Amazon API Gateway, and display the booking result through email notifications. These components together provide a complete user experience for the Event Ticket Booking System.
